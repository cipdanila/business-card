# 📇 Digital Business Card

A simple and clean digital business card created as part of the **freeCodeCamp** curriculum.

## 🎯 Project Goal
The goal of this project was to practice the basics of HTML structure and CSS styling by creating a personal business card layout.

## 🚀 Features
- **Clean Structure:** Uses a main container to hold all personal information.
- **Custom Styling:** Includes a specific background color, custom fonts, and centered alignment.
- **Social Links:** Includes functional links to GitHub, LinkedIn, and Twitter.

## 🛠️ Technologies Used
- **HTML5** (Images, paragraphs, links, and horizontal rules)
- **CSS3** (Width, padding, margins, and text alignment)

## 📝 What I Practiced
- Linking an external CSS file to an HTML document.
- Using **Class Selectors** to style specific elements.
- Centering a fixed-width element on the screen using `margin: auto`.
- Removing default link underlining with `text-decoration: none`.
